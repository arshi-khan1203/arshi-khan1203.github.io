Machine Learning Algorithms: A Visual Framework

Classification by Application Domain

Arshi Khan | 3SU2026 Model Development (AIML-501-01R3A) | Professor: John Wishart | May 2026

Introduction

This project explores the landscape of machine learning algorithms by organizing them according to their primary application domains and learning approaches. As artificial intelligence continues to expand across industries, understanding which algorithms work best for specific types of data and problems has become essential for effective model development. This visual framework provides a clear reference for algorithm selection based on whether you are working with tabular data, images, text, or generative applications.

Description

This visual framework categorizes 10 fundamental machine learning algorithms across four key domains: Tabular Data, Computer Vision, Natural Language Processing, and Generative AI. Each algorithm is classified by learning type (supervised or unsupervised) and includes real-world application examples.

Objective

The primary objective of this project was to create a clear, visually organized reference illustrating the relationship between machine learning algorithms and their practical applications. By mapping algorithms to specific domains and providing concrete use cases, this framework serves as a decision-making tool for selecting appropriate algorithms based on data type and problem requirements. Additionally, this project demonstrates my ability to synthesize technical concepts into accessible visual communication.

Process

The development process began with researching and selecting 10 algorithms that represent the core techniques used across major AI/ML domains. I analyzed each algorithm to determine its classification (supervised vs. unsupervised) and primary application areas. The algorithms were then organized by domain rather than by learning type, as this structure better reflects how practitioners approach real-world problems.

I created a multi-slide visual presentation that groups algorithms by their application domain, with each slide providing the algorithm name, learning type, brief explanation of how it works, and concrete examples. The final slide synthesizes key insights about algorithm relationships and selection criteria.

Algorithm Coverage

  • Tabular Data (5 algorithms): Linear Regression, Logistic Regression, Decision Trees, Random Forest, Support Vector Machines, K-Means Clustering
  • Computer Vision (2 algorithms): Convolutional Neural Networks (CNN), Support Vector Machines (SVM)
  • Natural Language Processing (2 algorithms): Recurrent Neural Networks (RNN), Transformer Models
  • Generative AI (2 algorithms): Transformer Models, Generative Adversarial Networks (GAN)

Value Proposition

Unique Value

This framework distinguishes itself by organizing algorithms according to practical application domains rather than purely theoretical categories. This approach mirrors how data scientists and ML engineers actually make algorithm selection decisions in practice. The visual format makes complex technical concepts accessible while maintaining accuracy, and the inclusion of real-world examples bridges the gap between theory and application.

Relevance

Understanding algorithm selection is fundamental to successful model development. This framework provides a practical reference for determining which algorithms to consider when faced with different types of data and problems. As machine learning continues to evolve, having a solid grasp of core algorithms and their appropriate use cases remains essential for anyone working in AI/ML development, data science, or related fields.
